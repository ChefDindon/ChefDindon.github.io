Snake

J'ai réalisé une version du jeu snake en python. Lorsqu'on exécute le programme une noouvelle fenêtre s'ouvre dans laquelle on peut jouer à snake à l'aide du module pyxel où il faut donc manger le plus de pommes sans toucher les bords ou se mordre la queue.

Domino

Avec l'aide d'un ami j'ai réalisé une version python du jeu des dominos. Pour créer le jeu nous avons utilisé la Programmation Orientée Objet en python qui nous a permis de créer les mains des joueurs,la pioche mais aussi de trier les dominos, de déterminer qui commence(celui qui a le plus grand domino double sinon celui qui a le domino avec le plus grand chiffre),de piocher et bien sur de gérer la partie.

Jeu de rôle

Avec ce même ami j'ai réalisé un jeu de rôle. Dans ce projet nous avon également utilisé la Programmation Orientée Objet en python afin de créer plusieurs héros ayant chacun des statistiques différentes(points de vie,dégats,vitesse) et différents monstres qui ont eux aussi des statistiques spécifiques à chacun. Le but du jeu est de progrésser dans un donjon où le héros doit affronter des monstres et plus le joueur monte dans les niveaux du donjon, plus les monstres sont forts.
